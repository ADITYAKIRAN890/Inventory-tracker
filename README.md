# Inventory Tracker

Inventory Tracker is a Flutter application designed to help you manage stock, monitor product quantities, and track inventory updates across devices.

## Features

- Track products and stock levels
- Add, edit, and remove inventory items
- View current inventory status at a glance
- Built with Flutter for cross-platform support

## Getting Started

These instructions will help you run the project locally.

### Prerequisites

- Flutter SDK installed
- Android Studio, Xcode, or a compatible IDE
- A connected device or emulator

### Install

1. Clone the repository:

```bash
git clone https://github.com/ADITYAKIRAN890/Inventory-tracker.git
cd Inventory-tracker
```

2. Fetch dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```

## Supported Platforms

- Android

- `lib/` — Main Flutter source code
